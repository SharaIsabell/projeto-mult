BÁSICO ESPECÍFICO DO CURSO

CPT01090 - BANCO DE DADOS

EMENTA

Conceitos Básicos: Arquitetura de um Sistema de Banco de Dados, Modelos de Dados, Linguagens de Definição e Manipulação de Dados. Modelagem de Dados. Modelos de Dados: Relacional, Hierárquicos e de Redes. Projeto de Banco de Dados Relacional: Dependência Funcional, Chaves, Normalização, Visões, Integração de Visões. Abrindo uma conexão com um SGBDR (JBDC). Executando comandos SQL. Transações. Otimização de Consultas. Controle de Concorrência. Controle de Integridade. Segurança./p>
